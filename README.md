#  Udemy Clone

A responsive **Udemy Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of the Udemy homepage with a clean, modern, and user-friendly interface. It was developed to strengthen my frontend development skills by practicing responsive layouts, Flexbox, Grid, and modern CSS styling techniques.

## 🚀 Features

*  Clean and modern UI inspired by Udemy
*  Fully responsive layout
*  Search bar with elegant design
*  Course recommendation section
*  Popular courses section
*  Course categories
*  Recommended topics section
*  Attractive course cards with hover animations
*  Professional footer layout
*  Smooth hover effects and transitions
*  Beginner-friendly project structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* CSS Flexbox
* CSS Grid
* Font Awesome
* Google Fonts
